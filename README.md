# Projeto Web Coding

Site desenvolvido para o projeto da faculdade.  
Cada membro da equipe é responsável por uma parte específica do site.

## Estrutura do projeto
- `src/`: código principal (HTML, CSS, JS)
- `assets/`: imagens, ícones, fontes, etc.
- `docs/`: documentação e arquivos de apoio.

## Como executar
Abra o arquivo `index.html` no navegador.
